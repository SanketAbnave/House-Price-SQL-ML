# 🏠 House Price Analysis (SQL + Machine Learning)

## 📌 Project Overview

This project analyzes house price data using SQL and builds a machine learning model to predict house prices.

---

## ⚙️ Technologies Used

* MySQL
* Python (Pandas, Scikit-learn)
* Jupyter Notebook

---

## 📊 SQL Analysis

* Average house price
* Price distribution by bedrooms
* Top locations by price
* Waterfront vs non-waterfront analysis
* Ranking houses using window functions

---

## 🤖 Machine Learning

* Model: Linear Regression
* Features:

  * bedrooms
  * bathrooms
  * sqft_living
* Evaluation: R² Score

---

## 📁 Project Structure

* dataset → raw data
* sql → SQL queries
* notebooks → ML model

---

## 🚀 How to Run

1. Run SQL file in MySQL
2. Import dataset into database
3. Open Jupyter Notebook
4. Run all cells

---

## 📈 Output

* SQL insights
* Predictive ML model

---

## 🎯 Conclusion

This project demonstrates end-to-end data analysis workflow combining SQL and Machine Learning.
